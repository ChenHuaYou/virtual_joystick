# Features
- Float Joystick
- Expansive area to interact with joystick

> **NOTE:** The color in area is only for make it visible

# Settings of this example
- VirtualJoystickAxis::Both
- VirtualJoystickType::Floating

# Run example
```sh
cargo apk run
```
